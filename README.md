Our database will cover four departments of the dealership which includes Sales, Finance, Service, and Parts.
Each department manager will have their own database to access their employees. The store manager will have a table that represents the overall scope of the dealership. 
The Service Department Manager will oversee the both Service and Parts Department. As of now, we are looking at having nine tables for this project.
